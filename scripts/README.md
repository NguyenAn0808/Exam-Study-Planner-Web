# Development Scripts

## Start Development Environment
```bash
# Full stack with Docker
npm run dev:docker

# Only database, run apps natively
npm run dev:db-only

# Clean restart
npm run dev:clean
```

## Build and Test
```bash
# Build production images locally
npm run build:images

# Run tests
npm run test:all
```

## Deployment
```bash
# Deploy to production (when Azure is ready)
npm run deploy:prod
```
